\begin{abstract}

El presente trabajo propone una alternativa para la producción de videojuegos en la forma de un framework que permita la producción de los mismos en un lenguaje funcional, en este caso el lenguaje Haskell, para así poder hacer uso de las ventajas que este tipo de lenguajes provee con el fin de mejorar la productividad y calidad del producto.

Con la realización de este trabajo se ha logró mostrar que un framework de juego programado en lenguaje Haskell puede ser usado para la producción de videojuegos, el framework creado provee una manera transparente para aprovechar el uso de concurrencia en procesadores modernos y la capacidad de expresar la lógica del juego en forma funcional.

\end{abstract}
